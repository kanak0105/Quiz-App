🧠 Interactive Quiz Engine
A sleek, responsive quiz application designed for a seamless user experience across all devices.

✨ Key Features
Dynamic Logic: Questions are injected via JavaScript for a fast, single-page feel.
Visual Feedback: Instant UI updates with color-coded validation (Green/Red).
Progress Tracking: Live score calculation and a final performance summary.
Smart Reset: Re-initialize the game state without refreshing the page.

🛠 Tech Stack
Frontend: HTML5, CSS3 (Flexbox/Grid)
Logic: Vanilla JavaScript (ES6+)
